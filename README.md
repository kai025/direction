## Usage

### Development

Just run and visit http://127.0.0.1:3000/

```bash
yarn
```

```bash
yarn dev
```

### Build

To build the App, run

```bash
yarn build
```

And you will see the generated file in `dist` that ready to be served.

### Deploy on Vercel

Go to Vercel and create a new Vite project, no extra setting needed
